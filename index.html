    • Container

    Container tags are the tags that contain some data such as text, image, etc. There are several container tags in HTML.

  • <div>

    <div> This id div  </div>

    • Span Tag

    span is a container for inline content

  • <span>

    <span> This is span tag  </span>

    • p Tag

    Paragraph tags are used for writing paragraphs in your website

  • <p>

    <p> content </p>

    • pre Tag

    pre tag represents pre-formatted text

  • <pre>

    <pre> content </pre>

    • code Tag

    code tag is used for representing source codes in your website

  • <code>

    <code> important java or python code </code>

    • Text Formating Tags are as followed

    Text formatting tags are used to format text or data of HTML documents. You can do certain things like creating italic, bold, strong text to make your document look more attractive and understandable.

  • <b>

    <b> Makes Text Bold </b>

  • <strong>

    <strong> Text that are important makes them italic </strong>

  • <i>

    <i> Makes text italic </i>

  • <em>

    <i> Makes text emphasized </i>

  • <sub>

    <sub> Makes text Subscrite </sub>

  • <sup>

    <sup> Makes text Superscrite </sup>

  • <ol> ordered list shows list itmes with number

    <ol>
  <li> 1.Data-1 </li>
  <li> 2.Date-2 </li>
  <li> 3.Date-3 </li>
</ol>

  • <ul> unordered list shows list itmes without number or shows with bullets points

    <ul>
  <li> • Data-1 </li>
  <li> • Date-2 </li>
  <li> • Date-3 </li>
</ul>
